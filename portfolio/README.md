# Portfolio QA Automation

## Objetivo

Este directorio reúne las evidencias generadas durante el desarrollo del proyecto **DemoBlaze Automation Framework**.

El objetivo es demostrar la aplicación de buenas prácticas de QA Automation utilizando Java, Selenium WebDriver, TestNG y Page Object Model.
